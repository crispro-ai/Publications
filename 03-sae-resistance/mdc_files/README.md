# MDC Files Archive

This directory contains all MDC (markdown cursor) files related to SAE resistance prediction work.

## Files Included

- `MANAGER_ROOT_CAUSE_ANALYSIS.mdc` - Root cause analysis of validation issues
- `PUBLICATION_PLAN_PROXY_VS_TRUE_SAE.mdc` - Publication planning documents
- `Scrap.mdc` - Scrap notes and temporary files
- Additional SAE-related MDC files from `.cursor/rules/sae/` and `.cursor/rules/MOAT/`

## Purpose

These files document the planning, strategy, and analysis process for SAE resistance prediction validation. They are archived here for reference and to keep the main publication directory clean.
